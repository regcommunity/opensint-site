# OpenSInt one-pager (GitHub Pages)

This folder is ready to publish as a GitHub Pages project site.

## Quick publish (project pages)

1. Create a new GitHub repo (example: `opensint-site`)
2. Copy these files to the repo root
3. Push to `main`
4. In GitHub: **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: `main` / `/ (root)`
5. Your site will be available at:
   - `https://<user-or-org>.github.io/opensint-site/`

## Optional custom domain

1. Add your domain in **Settings → Pages → Custom domain**
2. Add a `CNAME` file in the repo root with your domain (e.g. `opensint.org`)
3. Update DNS with your registrar
